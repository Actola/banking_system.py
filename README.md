# Simple Banking System

This is a console-based Simple Banking System developed using Python as part of the SEN 201 course assignment.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Exit the system

## Technologies Used
- Python
- Visual Studio Code

## How to Run the Program
1. Open the project folder in VS Code
2. Open the terminal
3. Run the command:
# banking_system.py
banking_system.py
